# Simple Python API

## Installation

Requires [Python](https://www.python.org/) 3.7+ to run.

Create virtual environment and activate it

```sh
python3 -m venv venv
source venv/bin/activate
```

Install the dependencies.

```sh
pip install -r requirements.txt
```

And start the server

```sh
uvicorn main:app
```
